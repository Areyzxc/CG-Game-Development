# Process 1: User Registration and Authentication

## Overview
This document outlines the process of user registration and authentication for the CG Game Development project.

## User Registration
1. The user navigates to the registration page.
2. The user fills out the registration form with the required fields:
   - Username
   - Email
   - Password
3. The user submits the form.

## User Authentication
1. The user navigates to the login page.
2. The user enters their username and password.
3. The system verifies the credentials.
4. Upon successful login, the user is redirected to their dashboard.
